# PHP常用函数系列之数组排序(一)

## Func List

| 函数名|排序依据|数组索引健保持|排序顺序|
|------|------|------------|-------|
|array_multisort()|值|键值关联的保持，数字类型的不保持|第一个数组或者由选项指定|
|sort()|值|否|升序|
|asort()|值|是|升序|
|rsort()|值|否|降序|
|arsort()|值|是|升序|
|ksort()|键|是|升序|
|krsort()|键|是|降序|
|shuffle()|值|否|随机|
|natcasesort()|值|是|自然排序，大小写不敏感|
|natsort()|值|是|自然排序|
|uasort()|值|是|自定义|
|uksort()|健|是|自定义|
|usort()|值|否|自定义|
